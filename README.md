# Bob Le's Portfolio

Bob Le's software engineer personal portfolio.

Live site: 
- http://thebuilderbob.com/
- https://bob-le-portfolio.vercel.app/

## Installation

Use the package manager [npm] to install project.

```bash
npm install
```

## Usage

Local Dev
```bash
npm run dev
```

Production Build
```bash
npm run build
```
